# 飽和水蒸気量シミュレータ (VaporApp-sim)

Rust と GUI クレート `egui` (`eframe`) を用いて作られた、デスクトップ向けのシンプルな飽和水蒸気量シミュレーションアプリです。
<img width="488" height="432" alt="スクリーンショット 2026-05-27 201940" src="https://github.com/user-attachments/assets/8c9ec99b-9bd3-4953-8df9-897443a3a32c" />

温度スライダーを動かすことで、その温度における**飽和水蒸気圧（hPa）**および**飽和水蒸気量（g/m³）**をリアルタイムに計算し、視覚的なゲージと共に表示します。

## ✨ 特徴

- **リアルタイム計算**: スライダーの変更が即座に画面に反映されます（イミディエイト・モード GUI）。
- **科学的アプローチ**: 飽和水蒸気圧の計算には一般的に広く用いられる **Tetens（テッテンズ）の式** を採用しています。
- **日本語対応**: 起動時にシステムフォント（MS ゴシック）を読み込むため、文字化け（豆腐文字）が起きません。

## 🛠️ 計算方法について

本アプリは以下の数式に基づいて設計されています。

1. **飽和水蒸気圧 $E(T)$ (hPa)**:
   $$E(T) = 6.1078 \times 10^{\frac{7.5T}{T + 237.3}}$$
2. **飽和水蒸気量 $a(T)$ (g/m³)**:
   $$a(T) = \frac{217 \times E(T)}{T + 273.15}$$

*(※ $T$ は摂氏温度 °C)*

## 🚀 動作方法

### 前提条件
- [Rust](https://www.rust-lang.org/) (Cargo) がインストールされていること。
- Windows 環境（デフォルト設定の場合）

### 実行手順

1. リポジトリをクローンまたはローカルに配置します。
2. プロジェクトのルートディレクトリで以下のコマンドを実行します。
